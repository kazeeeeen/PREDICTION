Data Understanding-Milk Quality

This dataset helps us to build machine learning models to predict the quality of milk.
This dataset consists of 7 independent variables ie pH, Temperature, Taste, Odor, Fat, Turbidity, and Color.
Defining each columns:
• pH - defines PH alus of the milk which ranges from 3 to 9.5 max : 6.25 to 6.90
• Temperature - defines Temprature of the milk which ranges from 34'C to 90'C max : 34'C to 45.20'C
• Taste - defines Taste of the milk which is categorical data 0 (Bad) or 1 (Good) max : 1 (Good)
• Odor - defines Odor of the milk which is categorical data 0 (Bad) or 1 (Good) max : 0 (Bad)
• Fat - defines Fat of the milk which is categorical data 0 (Low) or 1 (High) max : 1 (High)
• Turbidity - defines Turbidity of the milk which is categorical data 0 (Low) or 1 (High) max : 1 (High)
• Colour - defines Colour of the milk which ranges from 240 to 255 max : 255
• Grade - defines Grade (Target) of the milk which is categorical data Where Low (Bad) or Medium (Moderate) High
